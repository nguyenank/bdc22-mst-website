# bdc22-mst-website

An interactive webapp, [available here](https://highway-to-the-danger-zone.netlify.app/), showing off the Highway model created by Carlie Markey, Nayan Patel, An Nguyen, and Ben Howell.

![A screenshot of the web application.](./screenshot.png)

The Highway model and this webapp were created as part of a project for the Big Data Cup 2022. A writeup of the Highway model and the full source code for the project is [available here](https://github.com/nguyenank/bdc22-mst).
